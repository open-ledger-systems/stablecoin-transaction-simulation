# Flash USDT Demo Simulation

This file demonstrates **Flash USDT mock transactions** using Node.js and Python.  

## Overview
- Safe simulation of USDT transfers  
- Visualizes transaction flow in educational environment  
- No real USDT is used  

## Installation

### Node.js
```bash
npm install
node USDT_transaction_demo.js
pip install -r requirements.txt
python USDT_transaction_demo.py
